# ralph/ — ночной автономный цикл для piece amoCRM

Папка управляет автономным Ralph-циклом: каждая итерация — свежий headless-запуск `claude -p`,
агент берёт следующую невыполненную задачу из плана, делает её, верифицирует, отмечает чекбокс
и пишет запись в лог. Память между итерациями — только эти файлы + git.

- **prd.md** — план задач (T001…T028). Каждая задача самодостаточна: spec, files, pattern, verify.
  Порядок = порядок выполнения (риск/зависимости вперёд). Общие правила — в шапке файла.
- **activity.md** — лог итераций (дата/задача/изменения/команды/верификация/блокеры).
  Терминальные маркеры для скрипта: `<promise>COMPLETE</promise>` / `<promise>STUCK</promise>`.
- **PROMPT.md** — промпт одной итерации (контракт: выбор задачи, доки, скиллы, verify, коммит, журнал).
- **ralph.sh** — оркестратор: `./ralph/ralph.sh [MAX_ITER]` (default 25), ветка `feature/amocrm-piece`,
  гейт по лимитам подписки (снимок `~/.claude/rate-limits.json` + детект отказа CLI), логи в `ralph/logs/`.

Запуск на ночь: `caffeinate -dims ./ralph/ralph.sh 25`
